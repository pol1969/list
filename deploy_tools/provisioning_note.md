Обеспечение работы нового сайта
================================
## Необходимые пакеты:
* nginx
* Python 3.6
* virtualenv + pip
* Git

Например, в Ubuntu:

        sudo add-apt-repository ppa:fkrull/deadsnakes
        sudo apt-get install nginx git python36 python3.6-venv

## Структура папок:
Если допустить, что есть учетная запись пользователя в /home/username

        |--source
